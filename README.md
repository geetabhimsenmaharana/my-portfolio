## How to Run Locally

```bash
# Clone the repo
git clone https://github.com/geetabhimsenmaharana/my-portfolio.git

# Open in browser
open index.html
```

No build step needed — it's plain HTML.

---

## How to Deploy Updates

```bash
git add .
git commit -m "your message here"
git push origin main
```

GitHub Pages will automatically update within a minute.
